## Описание проекта

**Цель работы:** Разработать модель машинного обучения для бинарной классификации, которая предсказывает отказ оборудования (Target = 1) или его отсутствие (Target = 0) на основе технических параметров.

**Задачи:**
- Загрузка и предобработка данных
- Обучение нескольких моделей классификации
- Сравнение качества моделей
- Визуализация результатов
- Создание интерактивного веб-приложения для демонстрации
- Разработка презентации проекта

## Датасет

Используется датасет **"AI4I 2020 Predictive Maintenance Dataset"** из репозитория UCI Machine Learning.

### Характеристики датасета:
- **10 000** записей
- **14** признаков
- **Бинарная** целевая переменная
- **5** типов отказов

### Признаки:

| Название | Описание | Ед. измерения |
|----------|----------|----------------|
| Type | Тип продукта (L, M, H) | категориальный |
| Air temperature | Температура воздуха | K |
| Process temperature | Температура процесса | K |
| Rotational speed | Скорость вращения | rpm |
| Torque | Крутящий момент | Nm |
| Tool wear | Износ инструмента | min |

### Типы отказов:
- **TWF** - Износ инструмента
- **HDF** - Недостаточный теплоотвод
- **PWF** - Проблемы с мощностью
- **OSF** - Перегрузка
- **RNF** - Случайные отказы

## Метрики оценки

| Метрика | Описание | Формула | Интерпретация |
|:--------|:---------|:--------|:---------------|
| **Accuracy** | Доля правильных ответов | (TP + TN) / (TP + TN + FP + FN) | Чем ближе к 1, тем лучше |
| **Confusion Matrix** | Матрица ошибок | Таблица 2x2 | Показывает TP, TN, FP, FN |
| **ROC-AUC** | Площадь под ROC-кривой | ∫ ROC(t) dt | > 0.9 - отлично<br>> 0.8 - хорошо<br>> 0.7 - удовлетворительно |
| **Precision** | Точность | TP / (TP + FP) | Доля правильных положительных предсказаний |
| **Recall** | Полнота | TP / (TP + FN) | Доля найденных положительных объектов |
| **F1-score** | Гармоническое среднее | 2 * (Precision * Recall) / (Precision + Recall) | Сбалансированная метрика |

### Обозначения в матрице ошибок
- **TP (True Positive)** - верно предсказанный отказ
- **TN (True Negative)** - верно предсказанная нормальная работа
- **FP (False Positive)** - ложное срабатывание (предсказан отказ, но его нет)
- **FN (False Negative)** - пропуск отказа (предсказана норма, но отказ есть)
